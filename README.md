# Personal Finance & Expense Management System

## Objective
Build an Excel-based personal finance tracker that records income and expenses, compares actual spending with budgets, tracks savings performance, and visualizes monthly spending behavior through an interactive dashboard.

## Project Title
**Personal Finance & Expense Management System**

## Tools Used
- Microsoft Excel
- Pivot-ready structured dataset
- Formulas: `SUM`, `SUMIFS`, `IF`, `IFERROR`, `COUNTIFS`
- Charts and dashboard design
- GitHub for portfolio showcase

## Dataset Description
This project uses a realistic sample personal finance dataset with **179 transactions** across 12 months.

### Main Columns
- Date  
- Month  
- Income Source  
- Expense Category  
- Expense Type  
- Payment Mode  
- Amount  
- Income/Expense Indicator  
- Savings  
- Budget Allocated  
- Budget Remaining  
- Notes  

## Sheet Structure
1. **Raw Data** – source transactions
2. **Cleaned Data** – structured and formula-driven cleaned version
3. **Budget Planning** – month-category budget plan
4. **KPI Summary** – important business metrics
5. **Dashboard** – final executive view

## Features
- Monthly income and expense tracking
- Category-wise expense analysis
- Budget vs actual comparison
- Savings and savings rate calculation
- Payment mode analysis
- Fixed vs variable expense split
- Executive-style Excel dashboard
- Beginner-friendly project structure for portfolio use

## Key KPIs
- Total Income
- Total Expenses
- Net Savings
- Savings Rate %
- Budget Allocated
- Budget Remaining
- Budget vs Actual %
- Expense Transactions Count

## Key Insights
- Rent is the largest expense category.
- Fixed expenses form the majority of total spending.
- Bank Transfer and Credit Card are the most-used payment modes.
- Savings improve in high-income months with freelance, bonus, or business income.
- Budget tracking helps identify overspending months and categories quickly.
